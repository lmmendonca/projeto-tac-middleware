[Projeto] Forca com middleware - GUI

## Alunos

## Requisitos

- Nodejs 12.x
- NPM 6.x

## Instalação

Para instalar as dependências do client:

```bash
cd client/
npm install
```

## Iniciar aplicação

Para iniciar o client você precisa estar na subpasta `client/` e executar:

```bash
npm run start
```
